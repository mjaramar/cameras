Entrega para el módulo VISINF

Un estudio gráfico sobre las cámaras digitales del mercado

Como práctica para el módulo Visualización de la Información del máster Big Data & Data Science que se desarrolla en Sevilla se propone el análisis gráfico de un conjunto de datos de libre elección. Se recomienda el uso de la librería ggplot2.

El archivo comprimido (.rar) que se adjunta en la entrega incluye:

  • Dataset : cameras.csv
  • R Markdown : Entrega Visualización.Rmd
  • HTML : Entrega_Visualización.html
  • Gráficos png : Algunos gráficos generados en formato .pgn
